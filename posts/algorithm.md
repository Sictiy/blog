- 卡特兰数
  - 应用
    - 入栈出栈顺序：1 -1 1 1 -1 1 ... > 0 折线法证明结果为： c(2n, n-1)
    - 递归式满足：fn = f(1)*f(n-1) + f(2)*f(n-2)*...*f(n-1)*f(1) 可求解得出 fn = c(2n ,n) / n+1
      - 母函数（生成函数）
      - 二项式定理
      - 隔板法
  - 公式：fn = c(2n, n) / n+1 = c(2n, n) - c(2n, n-1)
  - 例题
    - 三角形问题

  - 树状数组
    - 从上往下建树，第一层为2^n,之后为空白处开始加2^n
    - 求和：二进制数每次去掉最后一个1，累加
      - 去掉最后一个1: a = x- (x & (-x))
    - 更新第x个数，则需要更新x对应的二进制数最后一个1代表数加该数
      - 求和结果为: a = x + ( x & (-x))
    - 建树：
      - 从第一个位置开始更新
      - 对1...n 的每个数:

      ```go
      j := i + ( i & -i )
      if i < n + 1 {
          bit[j] += bit[i]
      }

      ```
